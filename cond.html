Introduction

Les structures de contrôle conditionnelles en C vous permettent de prendre des décisions dans votre code. Cela signifie que vous pouvez exécuter différentes instructions en fonction des conditions que vous spécifiez.

Les principales structures conditionnelles en C sont :

  • Instruction if
  • Instruction else
  • Instruction else if
  • Instruction switch

Instruction if

L'instruction if permet d'exécuter un bloc de code si une condition est vraie. Voici un exemple :


#include <stdio.h>

int main() {
int age = 18;

if (age >= 18) {
printf("Vous êtes majeur.\n");
}
return 0;
}

Dans cet exemple, le message "Vous êtes majeur." s'affichera si la variable age est supérieure ou égale à 18.

Instruction switch

L'instruction switch est utilisée pour exécuter un ensemble d'instructions en fonction de la valeur d'une variable. Voici comment cela fonctionne :


#include <stdio.h>

int main() {
int jour = 3;

switch (jour) {
case 1:
    printf("Lundi\n");
    break;
case 2:
    printf("Mardi\n");
    break;
case 3:
    printf("Mercredi\n");
    break;
default:
    printf("Jour inconnu\n");
}
return 0;
}

Dans cet exemple, le programme affichera "Mercredi" car la variable jour a la valeur 3.

Exemples de décisions

Voici quelques exemples de décisions plus complexes que vous pouvez réaliser avec les instructions conditionnelles :

  • Vérification de l'égalité de deux nombres.
  • Évaluation de l'état d'une variable booléenne.
  • Exécution d'actions différentes selon la valeur d'une variable.

#include <stdio.h>

int main() {
int a = 5, b = 10;

if (a > b) {
printf("a est plus grand que b.\n");
} else if (a < b) {
printf("a est plus petit que b.\n");
} else {
printf("a est égal à b.\n");
}
return 0;
}
