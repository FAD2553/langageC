Introduction

Les fonctions en C sont des blocs de code qui effectuent des tâches spécifiques. Elles permettent de diviser un programme en sous-programmes, facilitant la réutilisation du code et la maintenance.

Définition des fonctions

Une fonction est un ensemble d'instructions qui peuvent être appelées à tout moment dans un programme. Les fonctions peuvent accepter des paramètres et retourner une valeur.

Exemples de types de fonctions :

  • Fonctions sans retour ni paramètres
  • Fonctions avec retour et sans paramètres
  • Fonctions avec paramètres et retour

Syntaxe des fonctions

La syntaxe générale d'une fonction en C est la suivante :


type_de_retour nom_de_fonction(paramètres) {
    // corps de la fonction
}

Voici un exemple d'une fonction simple :


#include <stdio.h>

int ajouter(int a, int b) {
    return a + b;
}

int main() {
    int resultat = ajouter(5, 3);
    printf("Le résultat est : %d\n", resultat);
    return 0;
}

Exemples de fonctions

Voici quelques exemples de fonctions courantes :

  • Fonction pour calculer la factorielle : 
    
int factorielle(int n) {
    if (n <= 1) return 1;
    return n * factorielle(n - 1);
}
  • Fonction pour vérifier si un nombre est pair : 
    
int estPair(int n) {
    return n % 2 == 0;
}
