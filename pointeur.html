Introduction

Les pointeurs sont des outils puissants qui permettent de gérer directement la mémoire en C. Ils sont comme des cartes au trésor qui vous guident vers les adresses de mémoire où sont stockées vos données.

Qu'est-ce qu'un Pointeur ?

Un pointeur est une variable qui contient l'adresse d'une autre variable. Voici quelques caractéristiques des pointeurs :

  • Ils permettent d'accéder à la mémoire de manière dynamique.
  • Ils sont utilisés pour passer des arguments par référence aux fonctions.
  • Ils sont essentiels pour manipuler des tableaux et des chaînes de caractères.

Utilisation des Pointeurs

Voici comment déclarer et utiliser un pointeur :


int main() {
    int var = 42;     // Variable
    int *ptr = &var; // Pointeur qui stocke l'adresse de var

    // Affichage des valeurs
    printf("La valeur de var: %d\n", var);
    printf("L'adresse de var: %p\n", (void*)&var);
    printf("La valeur du pointeur ptr: %p\n", (void*)ptr);
    printf("La valeur pointée par ptr: %d\n", *ptr);
    
    return 0;
}

Dans cet exemple :

  • Nous avons une variable var avec une valeur de 42.
  • Nous déclarons un pointeur ptr qui stocke l'adresse de var.
  • Nous affichons la valeur de var, son adresse, la valeur du pointeur, et la valeur pointée par ptr.

Exemples Pratiques

Voici quelques exemples d'utilisation des pointeurs :

  • Manipulation de Tableaux : Les pointeurs peuvent être utilisés pour parcourir des tableaux.
  • Passage par Référence : Passer des variables à une fonction pour les modifier directement.
  • Allocation Dynamique : Gérer la mémoire dynamique avec malloc et free.

Voici un exemple d'allocation dynamique :


#include <stdlib.h>

int main() {
    int *arr = malloc(5 * sizeof(int)); // Allocation dynamique d'un tableau de 5 entiers

    for (int i = 0; i < 5; i++) {
        arr[i] = i * 10; // Remplissage du tableau
        printf("%d ", arr[i]);
    }

    free(arr); // Libération de la mémoire
    return 0;
}

Dans cet exemple, nous utilisons malloc pour allouer de la mémoire pour un tableau d'entiers, puis nous le remplissons et affichons ses valeurs avant de libérer la mémoire avec free.
