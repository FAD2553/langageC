Définition des variables

Une variable est un espace de stockage nommé qui peut contenir une valeur. En C, les variables sont utilisées pour stocker des données temporaires qui peuvent être modifiées au cours de l'exécution d'un programme. Elles permettent de rendre le code dynamique et interactif.

Types de variables

En C, il existe plusieurs types de variables, chacun ayant ses propres caractéristiques :

  • int : Pour les entiers (ex: 42, -7)
  • float : Pour les nombres à virgule flottante (ex: 3.14, -0.001)
  • double : Pour les nombres à virgule flottante avec une plus grande précision.
  • char : Pour les caractères individuels (ex: 'a', 'Z')
  • bool : Pour les valeurs booléennes (true ou false).

Déclaration et initialisation

Pour déclarer une variable, vous devez spécifier son type suivi de son nom. Vous pouvez également l'initialiser lors de la déclaration. Voici un exemple :


int age = 25;
float hauteur = 1.75;
char initiale = 'K';

Dans cet exemple, nous déclarons une variable age de type int, une variable hauteur de type float, et une variable initiale de type char.

Exemples pratiques

Voici un exemple simple qui montre comment utiliser des variables dans un programme :


#include <stdio.h>

int main() {
    int age = 30;
    float hauteur = 1.80;

    printf("Vous avez %d ans et vous mesurez %.2f mètres.\n", age, hauteur);
    return 0;
}

Ce programme déclare deux variables, age et hauteur, et les affiche à l'écran.
