Définition des structures

Les structures en C sont des collections de variables de différents types regroupées sous un même nom. Elles permettent de créer des types de données personnalisés, facilitant ainsi l'organisation et la gestion des données complexes.

Déclaration d'une structure

Pour déclarer une structure, utilisez le mot-clé struct suivi du nom de la structure et des membres entre accolades. Voici un exemple :


struct Personne {
    char nom[50];
    int age;
    float taille;
};

Dans cet exemple, la structure Personne a trois membres : un tableau de caractères pour le nom, un entier pour l'âge et un flottant pour la taille.

Utilisation des structures

Après avoir déclaré une structure, vous pouvez créer des variables de ce type et accéder à leurs membres à l'aide de l'opérateur point (.). Voici un exemple :


struct Personne john;

strcpy(john.nom, "John Doe");
john.age = 30;
john.taille = 1.75;

printf("Nom: %s, Age: %d, Taille: %.2f\n", john.nom, john.age, john.taille);

Dans cet exemple, nous créons une variable john de type Personne et initialisons ses membres.

Exemples pratiques

Les structures sont souvent utilisées pour modéliser des données complexes. Par exemple, vous pouvez utiliser des structures pour représenter des livres, des étudiants ou des véhicules. Voici un exemple de structure pour un livre :


struct Livre {
    char titre[100];
    char auteur[50];
    int annee;
};

struct Livre monLivre;

strcpy(monLivre.titre, "Le Petit Prince");
strcpy(monLivre.auteur, "Antoine de Saint-Exupéry");
monLivre.annee = 1943;

printf("Titre: %s, Auteur: %s, Année: %d\n", monLivre.titre, monLivre.auteur, monLivre.annee);

Les structures vous aident à garder vos données organisées et à faciliter leur manipulation dans votre programme.
