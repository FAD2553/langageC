Histoire du C

Le langage C a été développé par Dennis Ritchie entre 1969 et 1973 aux Bell Labs. Initialement créé pour le développement du système d'exploitation UNIX, il est devenu l'un des langages de programmation les plus influents de l'histoire de l'informatique.

Points clés de l'histoire du C :

  • 1972 : Première version majeure du C
  • 1978 : Publication du livre "The C Programming Language" par Brian Kernighan et Dennis Ritchie
  • 1989 : Standardisation du C (ANSI C)
  • 1999 : Publication du standard C99

Caractéristiques du C

Le C est connu pour ses caractéristiques uniques qui en font un langage puissant et flexible :

  • Langage de bas niveau avec des fonctionnalités de haut niveau
  • Portabilité sur différentes plateformes
  • Efficacité et performance
  • Contrôle direct de la mémoire
  • Syntaxe concise

Ces caractéristiques font du C un excellent choix pour le développement système, les applications embarquées et les logiciels nécessitant des performances élevées.

Structure d'un programme C

Un programme C typique comprend les éléments suivants :

  • Directives de préprocesseur (ex: #include)
  • Fonction main()
  • Déclarations de variables
  • Instructions
  • Commentaires

Voici un exemple de programme C simple :


#include <stdio.h>

int main() {
    // Ceci est un commentaire
    printf("Hello, World!\n");
    return 0;
}

Compilation et exécution

Pour exécuter un programme C, vous devez d'abord le compiler. Voici les étapes de base :

Étapes pour compiler et exécuter un programme C avec CodeBlocks :

  • Télécharger et installer CodeBlocks (si vous ne l'avez pas déjà fait) : Vous pouvez télécharger le programme à partir de ce lien officiel.
  • Créer un nouveau projet : Ouvrez CodeBlocks. Cliquez sur "Create a New Project" dans le menu principal. Sélectionnez "Console Application", puis cliquez sur "Go". Choisissez C comme langage et suivez les étapes pour nommer et enregistrer votre projet.
  • Écrire votre code C : Une fois le projet créé, ouvrez le fichier source par défaut, généralement nommé main.c. Remplacez le contenu par le code que vous souhaitez exécuter, ou bien écrivez un nouveau programme en C.
  • Compiler le code : Dans la barre d'outils, cliquez sur "Build", ou appuyez sur F9. Cela va compiler votre programme. Si tout est correct, vous verrez un message indiquant "Build finished" dans la console en bas de l'interface de CodeBlocks.
  • Exécuter le fichier compilé : Après la compilation, cliquez sur "Run" ou appuyez sur Ctrl + F10. Cela exécutera votre programme et affichera les résultats dans la console de CodeBlocks.

Exemple : Imaginons que vous ayez un programme C simple qui imprime "Hello, World!". 


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
    }

Étape 1 : Vous placez ce code dans votre fichier source.


Étape 2 : Vous compilez en cliquant sur "Build".


Étape 3 : Vous exécutez en cliquant sur "Run", et la console affichera : Hello, World!
