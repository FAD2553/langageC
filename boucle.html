Introduction

Les boucles en C permettent d'exécuter un bloc d'instructions plusieurs fois en fonction d'une condition. C'est utile lorsque vous voulez répéter une tâche jusqu'à ce qu'une condition soit remplie.

Les trois principales boucles en C sont :

  • La boucle while
  • La boucle for
  • La boucle do-while

Boucle while

La boucle while exécute un bloc de code tant qu'une condition donnée est vraie. Voici un exemple :


#include <stdio.h>

int main() {
    int compteur = 0;

    while (compteur < 5) {
        printf("Compteur: %d\n", compteur);
        compteur++;
    }
    return 0;
}

Dans cet exemple, le programme affiche les valeurs de compteur tant que cette variable est inférieure à 5.

Boucle for

La boucle for est généralement utilisée lorsqu'on connaît à l'avance le nombre d'itérations. Voici un exemple :


#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {
        printf("i vaut : %d\n", i);
    }
    return 0;
}

Dans cet exemple, le programme affiche les valeurs de i de 0 à 4.

Boucle do-while

La boucle do-while est similaire à la boucle while, à la différence que la condition est vérifiée après chaque exécution du bloc de code, garantissant que le code s'exécute au moins une fois. Exemple :


#include <stdio.h>

int main() {
    int compteur = 0;

    do {
        printf("Compteur: %d\n", compteur);
        compteur++;
    } while (compteur < 5);

    return 0;
}

Dans cet exemple, le programme exécute le bloc de code une première fois, puis vérifie la condition.

Exemples supplémentaires

Voici quelques exemples plus complexes utilisant des boucles :

  • Itérer à travers un tableau d'éléments avec une boucle for.
  • Compter les occurrences d'un élément dans une liste avec une boucle while.
  • Répéter une action jusqu'à la satisfaction d'une condition utilisateur avec une boucle do-while.

#include <stdio.h>

int main() {
    int tableau[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        printf("Élément %d: %d\n", i, tableau[i]);
    }
    return 0;
}
